Welcome

I'm a web developer building clean and efficient websites.

My focus is building functional and efficient websites that are easy to use and maintain.

I enjoy building interesting programs to be showcased on the web. For example I recently did some open data visualization for Toronto's BikeShare system.

My main stack is centred around using Django to build sites that seamlessly serve data from custom Python programs. For example, the BikeShare visualization site pulls and processes API data to generate viewable data for the site. My proof of concept developer certification site even allows arbitrary Python code to be safely run on the server, and have the result shown on the site for testing purposes.

Alongside Django, my main stack includes Linux, PostgreSQL, and Nginx.

I release my work on GitHub with clear instructions and documentation for myself and others. If you have any questions about configuring or running any of my projects, feel free to contact me.

I've recently picked up an interest in visualizing open data such as Toronto's and Canada's open data offerings. If you use BikeShare/BIXI you may be interested in visualizing your BikeShare trips.
