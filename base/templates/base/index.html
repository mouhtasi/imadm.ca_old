Welcome

I'm a web developer specialized in building clean and efficient web applications.

My current focus is web development centred around using Django to build sites that seamlessly serve data from custom Python programs. For example my set of BikeShare Toronto Tools pulls and processes API data to generate viewable data on the site.

I'm very interested in creating beautiful yet efficient and functional websites that are easy to use and maintain. I release my work on GitHub with clear instructions and documentation. My experience in programming has allowed me to build interesting tools and applications, and using web frameworks like Django I can make them easily available online.

I've recently picked up an interest in visualizing open data such as Toronto's and Canada's open data offerings. If you use BikeShare/BIXI you may be interested in visualizing your BikeShare trips.
